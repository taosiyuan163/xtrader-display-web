直接在浏览器中打开 `index.html` 文件即可查看所有页面原型图。

## 📐 设计规范 (Soft Modern Mobile UI)

### 色彩系统

| 用途 | 色值 | 说明 |
|------|------|------|
| 背景色 | `#FAFAFA` | 浅灰色背景 |
| 主要文字 | `#1E1E1E` | 深色文字 |
| 深色卡片 | `#333D44` | 深炭灰色 |
| 蓝色强调 | `#93B8C9` | 柔和蓝色 |
| 浅绿卡片 | `#E8F5E8` | 淡绿背景 |
| 浅紫卡片 | `#F0EDF3` | 淡紫背景 |
| 金黄强调 | `#FFC300` | 品牌主色 |
| 收益正值 | `#22C55E` | 绿色 |
| 收益负值 | `#EF4444` | 红色 |

### 字体规范

- **字体族**: Inter, sans-serif
- **Display**: 32px / font-weight: 700 / line-height: 1.2
- **Heading**: 18px / font-weight: 600 / line-height: 1.3
- **Body**: 14px / font-weight: 400 / line-height: 1.5
- **Caption**: 12px / font-weight: 400 / line-height: 1.4

### 间距系统

- **基础单位**: 8px
- `s_base`: 8px
- `s_2x`: 16px
- `s_3x`: 24px
- `s_4x`: 32px
- **布局内边距**: 24px

### 圆角规范

- **卡片**: 16px
- **元素/按钮**: 12px
- **圆形头像**: 50%

### 阴影效果

- **卡片阴影**: `0px 2px 4px rgba(0, 0, 0, 0.08)`
- **浮动按钮阴影**: `0px 4px 10px rgba(0, 0, 0, 0.20)`

## 📄 页面结构

### 1. 启动页 (Splash Screen)
- 品牌Logo展示
- 应用名称和标语
- 加载动画

### 2. 登录页 (Login)
- 邮箱输入框
- 密码输入框（带显示/隐藏）
- 忘记密码链接
- 登录按钮
- 注册入口

### 3. 注册页 (Register)
- 返回按钮
- 邮箱、密码、确认密码输入
- 服务条款勾选
- 注册按钮

### 4. 首页仪表盘 (Dashboard)
- 顶部：用户信息 + 通知/设置图标
- 总资产卡片（深色背景）
  - 资产金额
  - 涨跌变化
  - 充值/提现按钮
- 快捷统计卡片（2列）
  - 本月收益
  - 运行中AI数量
- 当前持仓列表
- 底部导航栏

### 5. AI交易员页 (AI Traders)
- 页面标题 + 创建按钮
- 配置状态提示（已配置模型/交易所数量）
- 交易员卡片列表
  - 交易员头像和名称
  - AI模型 + 交易所标签
  - 运行状态
  - 收益统计
  - 查看详情/启停按钮

### 6. 交易员详情页 (Trader Detail)
- 返回按钮 + 标题 + 设置
- 收益曲线图（可切换时间范围）
- 账户概览（4格网格）
  - 总净值
  - 可用余额
  - 总收益
  - 收益率
- AI决策记录列表

### 7. 排行榜页 (Competition/Ranking)
- 标题 + LIVE标记
- 冠军卡片（金色渐变背景）
- 排行榜列表
  - 排名徽章
  - 交易员名称和配置
  - 收益率

### 8. 个人中心 (Profile)
- 用户头像和邮箱
- VIP标签
- 设置菜单
  - 安全设置
  - 消息通知
  - 深色模式（开关）
  - 语言选择
  - 帮助与FAQ
  - 联系客服
  - 关于XTrader
- 退出登录按钮

### 9. 创建交易员弹窗 (Create Trader Modal)
- 底部弹出式设计
- 表单字段
  - 交易员名称
  - 选择AI模型
  - 选择交易所
  - 策略模板
- 取消/创建按钮

## 🎨 设计要点

1. **移动端优先**: 所有页面针对垂直方向移动设备设计
2. **柔和现代风格**: 大圆角、轻阴影、高对比度
3. **一致性**: 所有页面保持统一的色调、圆角、间距
4. **金黄色品牌色**: 作为主要强调色贯穿全应用
5. **卡片式布局**: 信息模块化，易于阅读
6. **底部固定导航**: 4个主要入口（首页、AI交易、排行榜、我的）
